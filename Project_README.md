# One Piece Rating Prediction using Machine Learning

## Project Overview
This project focuses on predicting One Piece episode ratings using Machine Learning techniques. The dataset was analyzed, preprocessed, and used to train a Simple Linear Regression model for rating prediction.

## Objectives
- Analyze the One Piece dataset.
- Perform data preprocessing and cleaning.
- Train a Machine Learning model using Simple Linear Regression.
- Evaluate the model using performance metrics.
- Predict episode ratings based on the available data.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Jupyter Notebook

## Dataset
The dataset contains information related to One Piece episodes and their ratings. It was used for training and testing the prediction model.

## Machine Learning Workflow
1. Import the dataset.
2. Preprocess and clean the data.
3. Split the dataset into training and testing sets.
4. Train the Simple Linear Regression model.
5. Make predictions on test data.
6. Evaluate model performance using:
   - R² Score
   - Mean Absolute Error (MAE)
   - Root Mean Square Error (RMSE)

## Results
The model achieved high prediction accuracy and demonstrated the effectiveness of regression techniques for rating prediction tasks.
